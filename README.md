# Pra que serve?
Esse é um script completamente testado, que tem a finalidade de corrigir bugs, remover coisas desnecessárias como bloatwares e adicionar funções realmente úteis ao Windows. Para os técnicos de informática, isso salva muito em campo

- Execute como Administrador

> Inspirado em Steins;Gate. Todas as alterações podem ser desfeitas sem nenhum problema ou arquivo residual, é como voltar no tempo, como se nada tivesse acontecido.

Enjoy it!
